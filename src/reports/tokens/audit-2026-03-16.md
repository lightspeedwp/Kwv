## Token Naming Conventions

### **WordPress Alignment** ✅ **EXCELLENT**

**Naming Pattern:**
```css
/* WordPress Core */
--wp-preset-{category}-{name}: {value};

/* TWB Alias */
--twb-{category}-{variant}: var(--wp-preset-{category}-{name});
```

**Examples:**
```css
/* Font Families */
--wp-preset-font-family-serif: 'Playfair Display', serif;
--twb-font-serif: var(--wp-preset-font-family-serif);

/* Font Sizes */
--wp-preset-font-size-h1: clamp(2.4rem, 5vw + 1rem, 4.5rem);
--twb-text-h1: var(--wp-preset-font-size-h1);

/* Colors */
--wp-preset-color-paper: #f5efe4;
--twb-color-paper: var(--wp-preset-color-paper);

/* Spacing */
--wp-preset-spacing-40: 1rem;
--twb-spacing-4: var(--wp-preset-spacing-40);
```

**✅ Benefits:**
- WordPress theme.json export-ready
- Double-aliasing for component flexibility
- Core WP compatibility maintained
