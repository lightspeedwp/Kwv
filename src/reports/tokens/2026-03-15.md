## Critical Issue: Hardcoded Hex Colors

### Issue Details

**Severity:** 🔴 **CRITICAL**  
**Impact:** Cannot verify contrast, violates design token standards, breaks dark mode

**Color Mapping:**

| Hardcoded Hex | Should Be | Token Name |
|---------------|-----------|------------|
| `#DAA520` | Gold | `var(--twb-color-gold)` |
| `#2C1810` | Dark brown | `var(--twb-color-ink)` or create `--twb-color-brown` |
| `#111111` | Black | `var(--twb-color-ink)` or create `--twb-color-black` |
| `#8B0000` | Dark red | `var(--twb-color-plum)` or create `--twb-color-red` |
| `#F5F5DC` | Beige | `var(--twb-color-paper)` |
| `#FDF6E3` | Light beige | Create `--twb-color-paper-light` |
| `#F9F9F9` | Light gray | `var(--twb-color-bg-secondary)` |
| `#F5F5F5` | Light gray | `var(--twb-color-bg-secondary)` |
| `#999999` | Medium gray | `var(--twb-color-text-muted)` |

### 4. ⚠️ Inconsistent Token Naming

**Severity:** 🟠 **HIGH**  
**Impact:** Confusing which variables change in dark mode

**Current State:**
```css
/* Brand colors (never change) */
--twb-color-paper: #f5efe4;
--twb-color-ink: #1e1a17;

/* Semantic colors (change in dark mode) */
--twb-color-bg-primary: var(--twb-color-paper); /* light */
.dark {
  --twb-color-bg-primary: #1a1714; /* dark */
}
```

**Recommendation:** Separate brand from semantic

```css
/* Brand colors - never change, use --twb-brand-* */
--twb-brand-paper: #f5efe4;
--twb-brand-ink: #1e1a17;
--twb-brand-plum: #5a2d3b;

/* Semantic colors - change in dark mode, use --twb-color-* */
--twb-color-bg-primary: var(--twb-brand-paper); /* light */
.dark {
  --twb-color-bg-primary: var(--twb-brand-ink); /* dark */
}
```

### 5. ⚠️ No Token Coverage Report

**Severity:** 🟠 **HIGH**  
**Impact:** Can't track migration progress

**Solution:** Create automated token coverage checker

**Script to create:**
```bash
#!/bin/bash
# token-coverage.sh
# Scan TSX files for hardcoded hex colors
# Output: coverage percentage

echo "Scanning for hardcoded hex colors..."
TOTAL_FILES=$(find . -name "*.tsx" | wc -l)
VIOLATIONS=$(grep -r "bg-\[#\|text-\[#\|border-\[#" --include="*.tsx" | wc -l)
echo "Files: $TOTAL_FILES"
echo "Hardcoded colors: $VIOLATIONS"
echo "Coverage: $(( (TOTAL_FILES - VIOLATIONS) * 100 / TOTAL_FILES ))%"
```
