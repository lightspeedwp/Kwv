## Violation Breakdown by Pattern

### Pattern 1: Hardcoded Black Backgrounds

**Count:** 42 instances  
**Common Usage:** Heroes, overlays, modals, sections  
**Current:** `bg-black`, `bg-[#111111]`, `bg-[#2C1810]`, `bg-[#333]`  
**Fix:** `bg-[var(--twb-color-bg-inverse)]`

**Example:**
```tsx
// WRONG
<div className="bg-black text-white">

// RIGHT
<div className="bg-[var(--twb-color-bg-inverse)] text-[var(--twb-color-text-on-dark)]">
```

---

### Pattern 2: Hardcoded White Text

**Count:** 100+ instances  
**Common Usage:** Headers, footers, navigation, overlays  
**Current:** `text-white`, `text-white/80`, `text-white/60`  
**Fix:** `text-[var(--twb-color-text-on-dark)]`

**Example:**
```tsx
// WRONG
<nav className="text-white/80">

// RIGHT
<nav className="text-[var(--twb-color-text-on-dark)]">
```

---

### Pattern 3: Hardcoded Accent Colors

**Count:** 28 instances  
**Common Usage:** Buttons, badges, highlights  
**Current:** `bg-[#DAA520]`, `bg-[#C19B76]`, `bg-[#8B0000]`  
**Fix:** Use semantic accent tokens

**Example:**
```tsx
// WRONG
<button className="bg-[#DAA520] text-[#2C1810]">

// RIGHT
<button className="bg-[var(--twb-color-accent-gold)] text-[var(--twb-color-text-on-accent)]">
```

---

### Pattern 4: Hardcoded Light Backgrounds

**Count:** 12 instances  
**Common Usage:** Section backgrounds, cards  
**Current:** `bg-[#F9F9F9]`, `bg-[#F5F5DC]`, `bg-[#FFF8E7]`  
**Fix:** Use semantic bg tokens

**Example:**
```tsx
// WRONG
<section className="bg-[#F9F9F9]">

// RIGHT
<section className="bg-[var(--twb-color-bg-secondary)]">
```

---

### Pattern 5: Conditional Dark Styling

**Count:** 18 instances  
**Common Usage:** Sections with variant props  
**Current:** `variant === 'dark' ? 'text-white' : 'text-black'`  
**Fix:** Remove conditionals, use semantic tokens everywhere

**Example:**
```tsx
// WRONG
<p className={variant === 'dark' ? 'text-white' : 'text-[var(--twb-color-text-primary)]'}>

// RIGHT
<p className="text-[var(--twb-color-text-primary)]">
```

## Token Usage Guide

| Use Case | Current (Wrong) | Correct Semantic Token |
|----------|----------------|------------------------|
| **Main page background** | `bg-white` | `bg-[var(--twb-color-bg-primary)]` |
| **Card/elevated surfaces** | `bg-white` | `bg-[var(--twb-color-bg-tertiary)]` |
| **Alternate sections** | `bg-[#F9F9F9]` | `bg-[var(--twb-color-bg-secondary)]` |
| **Dark sections in light mode** | `bg-black` | `bg-[var(--twb-color-bg-inverse)]` |
| **Primary body text** | `text-black` | `text-[var(--twb-color-text-primary)]` |
| **Text on dark backgrounds** | `text-white` | `text-[var(--twb-color-text-on-dark)]` |
| **Text on colored buttons** | `text-white` | `text-[var(--twb-color-text-on-accent)]` |
| **Secondary text** | `text-gray-600` | `text-[var(--twb-color-text-secondary)]` |
| **Primary CTA background** | `bg-[#5a2d3b]` | `bg-[var(--twb-color-accent-plum)]` |
| **Gold accent** | `bg-[#DAA520]` | `bg-[var(--twb-color-accent-gold)]` |
| **Borders** | `border-gray-300` | `border-[var(--twb-color-border-primary)]` |
| **Overlays** | `bg-black/50` | `bg-[var(--twb-color-bg-inverse)]/50` |
