  - Global reset: `animation-duration: 0.01ms !important`
  - Global reset: `transition-duration: 0.01ms !important`
  - Scroll behavior: `scroll-behavior: auto !important`
  - Radix UI animations: Disabled via `[data-state]` selectors
  - Tailwind animations: Disabled via class selectors
  - Updated JSDoc to document WCAG 2.3.3 compliance

**Prefers-Reduced-Motion Behavior:**

**What Gets Disabled:**
- ✅ All `animation` properties (set to 0.01ms, effectively instant)
- ✅ All `transition` properties (set to 0.01ms, effectively instant)
- ✅ Smooth scroll behavior (changed to `auto`)
- ✅ Radix UI modal/dialog slide/fade animations
- ✅ Accordion expand/collapse animations
- ✅ Dropdown menu slide animations
- ✅ Hover scale/shadow transitions
- ✅ Page transition effects

**What Gets Preserved:**
- ✅ Focus indicators (required for keyboard navigation)
- ✅ Color changes (instant, not motion-based)
- ✅ Opacity changes (instant, safe for motion sensitivity)
- ✅ Functionality (all features still work, just no motion)
- ✅ State changes (buttons still respond, just instantly)

**CSS Implementation:**
```css
@media (prefers-reduced-motion: reduce) {
  /* Global reset */
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }

  /* Radix UI animations */
  [data-state="open"],
  [data-state="closed"],
  .animate-in,
  .animate-out {
    animation: none !important;
    transition: none !important;
  }

  /* Preserve color changes (instant) */
  a, button, input, select, textarea {
    transition-property: color, background-color, border-color, opacity;
    transition-duration: 0.01ms !important;
  }
}
```
